Запустите Visual Studio.
Выберите меню Инструменты и затем выберите в нем пункт Внешние инструменты.
В диалоговом окне Внешние инструменты нажмите кнопку Добавить. Появится новый элемент.
Введите заголовок для нового пункта меню, например Command Prompt.
В поле Команда укажите файл, который должен запускаться, например %comspec% или C:\Windows\System32\cmd.exe.
В поле Аргументы укажите, где найти конкретную командную строку, которую вы хотите использовать, например /k "C:\Program Files (x86)\Microsoft Visual Studio\2017\Enterprise\Common7\Tools\VsDevCmd.bat" (эта команда запускает командную строку разработчика, установленную с Visual Studio 2017 Enterprise). Измените это значение в соответствии с вашей версией, выпуском и расположением установки Visual Studio.
Выберите значение для поля Исходный каталог, например Каталог проекта.
Нажмите кнопку ОК .
Новый элемент добавляется в меню, и вы можете вызывать командную строку из меню Инструменты.

*Инструменты=Средства
